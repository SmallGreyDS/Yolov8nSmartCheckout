YOLOv8 Smart Checkout System
This project is an AI-powered smart checkout system that utilizes the YOLOv8 (You Only Look Once) object detection model to identify items in real-time. It is designed to streamline the retail experience by automating the identification of products for a faster checkout process.

For a full overview of the project's goals, design, and implementation, you can view the presentation here:
View Presentation on Canva: https://canva.link/qjbzhlcuwets5ri

🚀 Features
Real-time Detection: High-speed object detection using the YOLOv8n (nano) model.

Automated Billing: Identify multiple items simultaneously to calculate totals.

Python-based UI: Easy-to-use interface for monitoring detections and transactions.

🛠️ Prerequisites
Before running the project, ensure you have Python installed. You will also need to install the following dependencies:

Bash
pip install ultralytics opencv-python
📂 Project Structure
main.py: The core Python script containing the detection logic and UI.

best.pt: The trained YOLOv8 weight file tailored for checkout items.

requirements.txt: List of necessary Python libraries.

💻 How to Run
Clone the repository:

Bash
git clone https://github.com/SmallGreyDS/Yolov8nSmartCheckout.git
Navigate to the folder:

Bash
cd Yolov8nSmartCheckout
Run the application:

Bash
python main.py
